# Реализация базовых подпрограмм линейной алгебры
Параллельные и последовательные вычисления на C++ с OpenMP.

## Задания
1. **Скалярное произведение векторов** — параллельный цикл (16 потоков)
2. **Умножение матриц** — тройной цикл, замер времени
3. **Решение СЛАУ с верхнетреугольной матрицей** — обратный ход
4. **Комбинация векторов** — α·x + y
